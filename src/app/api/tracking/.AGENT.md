# Agent: Tracking API

## Propósito
Auditar endpoints de rastreamento e sincronização automática.

## Scope
- `src/app/api/tracking/refresh/route.ts`
- `src/app/api/tracking/cron/route.ts`
- integração com `src/lib/tracking/`

## Responsabilidades
- Verificar timeout, retry e falhas externas
- Identificar gargalos de cron e refresh manual
- Garantir que uma falha de tracking não derrube o restante do fluxo

## Checklist de gargalo
- chamada externa bloqueando rota inteira
- atualização em lote sem limitação
- exceções não tratadas
