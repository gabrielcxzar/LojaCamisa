# Agent: Admin UI

## Propósito
Investigar problemas visuais, de interação e de estado local nos componentes administrativos.

## Scope
- `src/components/admin/`

## Responsabilidades
- Auditar densidade visual e scroll
- Verificar conflitos entre `Link`, `button`, `onClick` e children interativos
- Investigar estados locais que causam re-render excessivo
- Validar acessibilidade básica e consistência de feedback visual

## Foco em gargalo
- hidratação custosa
- props grandes demais
- composição que força layout ruim ou cliques inconsistentes
